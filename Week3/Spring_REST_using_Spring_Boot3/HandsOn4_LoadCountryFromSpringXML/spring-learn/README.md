\# HandsOn4 - Spring Core Load Country from Spring Configuration XML



\## Objective

Load Country bean using Spring XML configuration.



\## Technologies Used

\- Java

\- Spring Boot 3

\- Spring Context

\- Maven



\## Features

\- Bean configuration using XML

\- Load bean using ApplicationContext

\- Display country details

\- Logging of constructor, setters and getters



\## Output

Country : Country{code='IN', name='India'}
