\# Hands-On 5 - Generate JWT Token



\## Objective

Implemented an authentication service that generates a JWT token after successful Basic Authentication.



\## Technologies Used

\- Java 17

\- Spring Boot 3

\- Spring Security

\- JWT (JJWT)

\- Maven



\## Features

\- Configured Spring Security

\- Created Authentication Controller

\- Generated JWT token

\- Tested using Browser/Postman



\## Output

The `/authenticate` endpoint returns a JWT token after successful authentication.
