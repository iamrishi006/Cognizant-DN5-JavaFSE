\# Exercise 1: Mocking and Stubbing



\## Objective



To learn Mockito Mocking and Stubbing concepts by creating a mock object for an external API and returning predefined values.



\## Technologies Used



\* Java 17

\* JUnit 5

\* Mockito 5

\* Maven



\## Project Structure



src/main/java/org/example



\* ExternalApi.java

\* MyService.java



src/test/java/org/example



\* MyServiceTest.java



\## Implementation



1\. Created a mock object of ExternalApi using Mockito.

2\. Stubbed the getData() method to return "Mock Data".

3\. Injected the mock object into MyService.

4\. Verified the returned value using JUnit assertions.
