\# Exercise 1 - Setting Up JUnit



\## Objective



The purpose of this exercise is to set up JUnit 5 in a Maven project and execute a simple unit test.



\## Tools Used



\- Java 24

\- Maven 3.9.16

\- JUnit 5

\- IntelliJ IDEA



\## Project Structure



\- Calculator.java

\- CalculatorTest.java

\- pom.xml



\## Description



A simple Calculator class was created with an add() method.



A JUnit test case was written in CalculatorTest.java to verify that the add() method returns the correct result.



\## Test Case



Input:

\- 5

\- 5



Expected Output:

\- 10



\## Result



The test case executed successfully and all tests passed.



\## Learning Outcome



\- Learned how to create a Maven project.

\- Learned how to add JUnit dependency using pom.xml.

\- Learned how to write and run a basic unit test.

\- Understood the purpose of the @Test annotation and assertEquals() method.
