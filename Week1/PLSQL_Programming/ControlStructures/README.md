\# PL/SQL Programming - Control Structures



This folder contains solutions for Exercise 1 of the PL/SQL Programming module.



\## Files



\### DatabaseSetup.sql

Creates sample Customers and Loans tables and inserts test data.



\### CustomerInterestDiscount.sql

Applies a 1% discount on loan interest rates for customers above 60 years of age.



\### CustomerVIPUpdate.sql

Updates customer VIP status based on account balance.



\### LoanDueReminder.sql

Displays reminder messages for loans due within the next 30 days.
